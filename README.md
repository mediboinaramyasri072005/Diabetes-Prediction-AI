# 🧠 AI-Based Diabetes Prediction System

This project is a Machine Learning-based system that predicts whether a person is diabetic or not using health-related data.

## 📌 Project Description
Diabetes is a serious health condition that affects many people. This project uses Artificial Intelligence to help in early detection by analyzing medical attributes.

## 🎯 Objective
- Predict diabetes (Diabetic / Non-Diabetic)
- Use Machine Learning algorithms
- Help in early diagnosis

## 📊 Dataset
The dataset used is `diabetes_binary_health.csv` which includes:
- Age
- BMI
- Glucose Level
- Blood Pressure
- Physical Activity
- Other health indicators

Target:
- Diabetes_binary (0 = No, 1 = Yes)

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🤖 Model Used
Random Forest Classifier

## 🔄 Project Workflow
1. Data Collection
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Prediction
6. Evaluation

## 📈 Output
The system predicts whether a person is:
- Diabetic
- Non-Diabetic
